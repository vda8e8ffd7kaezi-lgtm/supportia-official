About Supportia

Supportiaとは？

「Support（支える）」＋「ia（人々・場所）」

看護師一人ひとりが支え合い、育ち合う場をつくる。
そんな想いを込めて、Supportiaは生まれました。

支え、育て、看護の世界に変化を…

その一歩を、あなたと一緒に踏み出したいと思っています。

Tagline

病む人も、働く人も、安心して生きられる社会をつくる。

看護師の未来に、やさしさと強さを。

支え、育て、看護の世界を変える。

ミッション・ビジョン・バリュー Mission / Vision / Value

Mission

ミッション

看護師に寄り添い、
力を育て、世界を優しくする。

Supportia は、しんどさを抱える看護師に寄り添い、
必要な力を身につけてもらいながら、
看護師を取り巻く世界そのものを優しく、
働きやすくしていくことを目指します。

Vision

ビジョン

看護師が
笑顔で働ける環境をつくり、
病む人の幸せにつなげる。

  • 〇 看護師自身が成長できるサポート
    → 誇りとやりがいを持てる
  • 〇 訪問看護の開業・運営サポート
    → 負担を軽減し、持続可能な仕組みづくり
  • 〇 接遇・社会人基礎力の向上
    → 看護師の地位や働きやすさUP
Value

バリュー

  • Compassion（思いやり）
    – 心からの支援を提供する
  • Empowerment（成長の支援）
    – キャリアや人生に自信を持てるようにする
  • Sustainability（持続可能な環境）
    – 長期的な支援を提供する
  • Collaboration（つながり）
    – 医療・介護業界と連携し未来を創る
  • Integrity（誠実さ）
    – 信頼される支援を提供する
